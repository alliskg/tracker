# Macro Tracker PWA

A personal, local-only daily nutrition tracker. Install on your iPhone as a PWA.

## Setup & Deploy

```bash
# 1. Install dependencies
npm install

# 2. Test locally
npm run dev
# Opens at http://localhost:5173/macro-tracker/

# 3. Deploy to GitHub Pages
npm run deploy
# Your app will be at https://YOUR_USERNAME.github.io/macro-tracker/
```

## Install on iPhone

1. Open your GitHub Pages URL in **Safari**
2. Tap the **Share** button (square with arrow)
3. Tap **"Add to Home Screen"**
4. Tap **Add**

The app runs entirely in your browser — no data leaves your phone.
